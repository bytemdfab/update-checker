Управление производственным предприятием для Украины, редакция 1.3, версия 1.3.35.2


Внимание! Текущая версия конфигурации "Управление производственным предприятием для Украины" предназначена для использования с версией технологической платформы 1С:Предприятие 8 не ниже 8.2.19.68.

Внимание! При обновлении конфигурации с версий ниже 1.3.15.6 обязательно:
- обновить конфигурацию до версии 1.3.15.6
- запустить систему 1С:Предприятие в режиме «Предприятие» под пользователем с ролью «Полные права» для того, чтобы выполнились обработки обновления данных с текущей версии до 1.3.15.6.
- продолжать обновление

Внимание! В конфигурации отсутствует украинский интерфейс.
